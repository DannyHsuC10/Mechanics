# Special Angle
$$\theta = |\tan^{-1}(\frac{1+(\sqrt{((\frac{r_3}{r_2})^2-1})\tan(\sqrt{((\frac{r_3}{r_2})^2-1})}{\tan(\sqrt{((\frac{r_3}{r_2})^2-1})-\sqrt{((\frac{r_3}{r_2})^2-1})})|$$

$$r_2 = r_3\cos(\frac\pi9)$$

$$\theta = |\tan^{-1}(\frac{1+(\sqrt{((\frac{r_3}{r_3\cos(\frac{\pi}{9})})^2-1})\tan(\sqrt{((\frac{r_3}{r_3\cos(\frac{\pi}{9})})^2-1})}{\tan(\sqrt{((\frac{r_3}{r_3\cos(\frac{\pi}{9})})^2-1})-\sqrt{((\frac{r_3}{r_3\cos(\frac{\pi}{9})})^2-1})})|$$

$$\theta = |\tan^{-1}(\frac{1+(\sqrt{\csc^2\frac{\pi}{9}-1})\tan(\sqrt{\csc^2\frac{\pi}{9}-1})}{\tan(\sqrt{\csc^2\frac{\pi}{9}-1})-\sqrt{\csc^2\frac{\pi}{9}-1}})|$$

$$\csc^2\theta-1 = \cot^2\theta$$

$$\theta = |\tan^{-1}(\frac{1+(\sqrt{\cot^2\frac{\pi}{9}})\tan(\sqrt{\cot^2\frac{\pi}{9}})}{\tan(\sqrt{\cot^2\frac{\pi}{9}})-\sqrt{\cot^2\frac{\pi}{9}}})|$$

$$\theta = |\tan^{-1}(\frac{1+(\cot\frac{\pi}{9})\tan(\cot\frac{\pi}{9})}{\tan(\cot\frac{\pi}{9})-\cot\frac{\pi}{9}})|$$

* $\cot(\frac{\pi}{9})$ is approximately $2.747$

$$\theta = |\tan^{-1}(\frac{1+(2.747)\tan(2.747)}{\tan(2.747)-2.747})|$$

* $\tan(2.747)$ is approximately $-0.416$

$$\theta = |\tan^{-1}(\frac{1-1.143}{-0.416-2.747})|$$

$$\theta = |\tan^{-1}(\frac{-0.143}{-3.163})|$$

$$\theta = |\tan^{-1}(0.045)|$$

$$\theta \simeq 0.045$$
