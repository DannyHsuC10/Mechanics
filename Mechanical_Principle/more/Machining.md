# Machining
## Cutting
### Cutting Speed
$$v = \frac{\pi DN}{1000}$$
### Taylor's formula 
* constant : $n,C$
$$vt^n = C$$
## Feeding
* feed rate : $f$
* total feed : $S$
$$S = Nf$$
### Multi-edge feeding
* edge : $e$
* feed per edge : $d$
$$S = Nde$$
### Linear feeding
* Length : $L$
* Cutting time ratio : $T = \frac35$
$$v = \frac{LN}{1000T} = \frac{LN}{1000\times\frac35 = 600}$$
### Centerless grinder feeding
$$v\sin\theta = \frac{\pi DN}{01000}\sin\theta$$
## Roughness
* Maximum roughness : $R_z$
* Average roughness : $R_a$
* Nose radius : $r$
$$R_z = \frac{f^2}{8r}$$

$$R_a = \frac14R_z$$

$$R_a = \frac{f^2}{8r}\times \frac14 = \frac{f^2}{32r}$$
## Taper
* Taper : $T$
* offset : $s$
$$T = \frac{D-d}{L} = \frac{2s}L$$
### Deflection angle
* $1:5 >> 5.73^o$
$$\tan\theta \simeq \theta = \frac{D-d}{2L}\frac{180}{2\pi}$$
## Thread production
### thread cutting
* Number of separations : $N$
* Separation rate : $n$
* Lead : $L$
$$\frac{Sn}L = N$$
### Metric thread feeding
* pitch : $P$
$$H = P\cdot\sqrt3\cdot\frac12\frac68\simeq0.6495P\simeq0.65P$$
* Straight feed (Roughest)
$$S = 0.65P$$
* Vertical feed (smooth)
$$S = 0.65P$$
* Oblique feed (most fine)
$$S = P\frac68 = 0.75P$$
### Thread lead
$$\frac{T_\text{spindle gear}}{T_\text{Lead Screw}} = \frac{L_\text{Workpiece}}{L_\text{Lead Screw}}$$
## Gear Manufacturing
### Dividing method
* thread number : n
$$\frac{T_\text{Dividing head}}{n_\text{Worm}} = T_\text{Workpiece}$$
### Dividing angla
* Angle between two teeth : $X$
$$X^o = n_\text{Worm}\times\frac{360^o}{T_\text{Dividing head}}$$
## Welding
### Ohm's law
$$V = IR$$
### Joule's law
$$Q = I^2Rt = Pt$$
## Vernier caliper
* Accuracy : $Acc$
$$Acc = 1-\frac{\text{scale}}{\text{divided quantity}}$$
