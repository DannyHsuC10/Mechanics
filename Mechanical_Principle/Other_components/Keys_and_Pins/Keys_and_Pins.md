# Keys and Pins

## Unit conversion and Basic calculations

* RPM
$$\omega = \frac{2\pi N}{60}$$
* work
$$W = T\theta = Fx$$
$$P = \tau\omega = Fv$$
* horsepower

$$1\text{PS} = 75\text{kgf}\cdot\text{m/sec} = 735\text{watt}$$

## Symbols

* Force : $F$
* Torque : $T$
* Shaft diameter : $D$
* Width : $W$
* Length : $L$
* height : $H$
* Area : $A$
* shear stress : $\tau$
* compressive stress : $\sigma$

## Key load

$$\sigma = \frac{F}{A_\sigma}$$

$$\tau = \frac F{A_\tau}$$

$$F = \frac{2T}{D}$$

$$\sigma = \frac{F}{A} = \frac{2T}{D}/(\frac H2L) = \frac{4T}{DHL}$$

$$\tau = \frac{F}{A} = \frac{2T}{D}/(WL) = \frac{2T}{DWL}$$

$$T = \frac{\sigma DHL}{4} = \frac{\tau DWL}{2}$$

$$\frac HW = \frac{2\tau}{\sigma}$$

## Pin load

$$A = \pi R^2 = \pi(\frac D2)^2 = \frac{\pi D^2}{4}$$

$$\frac{F}{A} = \frac{F}{\frac{\pi D^2}{4}} = \frac{4F}{\pi D^2}$$
