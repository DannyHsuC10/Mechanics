# Spring
 ## Diameter
* Outer diameter : $D_o$
* inner diameter : $D_i$
* Middle diameter : $D_m$
* Wire diameter : $d$
$$D_m  = \frac{D_i+D_o}2$$

$$D_m = D_o-d$$

$$D_m = D_i+d$$

$$d = \frac{D_o-D_i}2$$
## Spring index and spring constant
* Spring index : $C$
* spring constant : $k$

$$C=  \frac{D_m}{d} = \frac{D_o}{d}-1 = \frac{D_i}{d}+1$$

$$k = \frac Fx$$
## series and parallel Springs
### In Series
* **2 spring**
$$k_\text{total} = \frac{k_1k_2}{k_1+k_2}$$
* **n spring**
$$k_\text{total} = \frac1{\frac1{k_1}+\frac1{k_2}+\frac1{k_3}+...\frac1{k_n}} = (\sum\frac1{k_i})^{-1}$$
### In Parallel
* **2 spring**
$$k_\text{total} = k_1+k_2$$
* **n spring**
$$k_\text{total} = k_1+k_2+k_3+...k_n = \sum k_i$$
