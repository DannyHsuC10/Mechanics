# Brakes
## Block brake
<div style="text-align: center;">
<img src="https://hackmd.io/_uploads/Hyzijozuel.png" alt="image"width="300">
</div>

$$\tau = fr = \mu Nr$$

$$\sum M_o = 0\qquad Na-fb-Fl = 0$$

$$\frac{\tau}{\mu r}a-\frac{\tau}{r}b-Fl = 0$$

$$F = \frac{\tau(a-\mu b)}{\mu rl}$$

**reverse**
$$F = \frac{\tau(a-\mu b)}{\mu rl}$$
* if $a-\mu b<0$, self lock
## Band brake
<div style="text-align: center;">
<img src="https://hackmd.io/_uploads/HkKnisMuee.png" alt="image"width="300">
</div>

$$\frac{F_1}{F_2} = e^{\mu\theta}$$

$$\tau = (F_1-F_2)r$$

$$F_1 = \frac{\tau e^{\mu\theta}}{F(e^{\mu\theta}-1)}$$

$$F_2 = \frac{\tau}{r(e^{\mu\theta}-1)}$$

$$\sum M_o = 0\qquad F_2a-Fl = 0$$

$$\frac T{r(e^{\mu\theta}-1)}a-Fl = 0$$

$$F = \frac{Ta}{rl(e^{\mu\theta}-1)}$$

**reverse**
$$F = \frac{Tae^{\mu\theta}}{rl(e^{\mu\theta}-1)}$$

## Disc brake
<div style="text-align: center;">
<img src="https://hackmd.io/_uploads/ryN0ojz_eg.png" alt="image"width="300">
</div>

$$D_m = \frac{D_o+D_i}{2}$$

$$\tau = f\frac{D_m}2 = \mu F\frac{D_m}2$$

$$F = \frac{2\tau}{\mu D_m}$$
## Brake cooling

## Braking power
$$\sigma = \frac FA$$

$$F = \sigma A$$

$$f = \mu F = \mu\sigma A$$

$$P = fv = \mu\sigma Av$$
