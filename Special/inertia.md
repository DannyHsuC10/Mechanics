# inertia

## inertia

* The ability of an object to maintain its state of motion
* the property of resisting changes in momentum.

## relative inertia

$$v = cv'$$

$$\frac12mv^2 = \frac12m'v'^2$$

$$m' = \frac m{c^2}$$

## reduced mass

$$\mu = \frac1{\frac1{m_A}+\frac1{m_B}} = \frac{m_Am_B}{m_1+m_2}$$

$$F_{AB} = m_Aa_A$$

$$F_{BA} = m_Ba_B$$

$$F_{AB}+F_{BA} = 0$$

$$m_Aa_A+m_Ba_B = 0$$

$$a = a_A-a_B = (1+\frac{m_A}{m_B})a_A$$

$$= \frac{m_A+m_B}{m_Am_B}m_Aa_A = \frac{F}{\mu}$$

## (Fictitious force)(pseudo force)

$$\sum F_{real} = ma$$

### Accelerated coordinate system

$$\sum F_{real}+F_{fict} = ma$$

* Centrifugal force
$$F_{cen} = m\omega^2r$$
* Coriolis force
$$F_{cor} = 2m\omega v$$
* Euler force
$$F_E = m\alpha r$$

### Fictitious force system

$$\sum  F_{real}+(ma_{fict}+2mv\omega+m\omega^2r+m\alpha r) = ma_{real}$$
