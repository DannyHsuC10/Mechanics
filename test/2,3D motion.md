# 2,3D motion
position vector $= \vec{r}$

$$\vec{r} = x\vec{i}+y\vec{j}+z\vec{k}$$

## Moving distance
$$r = \sqrt{x^2 + y^2 + z^2}$$

$$Î”\vec{r} = \vec{r_f}-\vec{r_i}$$

## Velocity
$$v = \frac{dr}{dt}$$

$$\vec{v}= v_x\vec{i}+v_y\vec{j}+v_z\vec{k}$$
:::success
$$vx = \frac{dx}{dt}\qquad vy = \frac{dy}{dt}\qquad vz = \frac{dz}{dt}$$
:::
## Acceleration
$$\vec{a_{avg}} = \frac{\vec{v_f}-\vec{v_i}}{t_f-t_i} = \frac{Î”\vec{v}}{Î”t}$$

$$\vec{a}= a_x\vec{i}+a_y\vec{j}+a_z\vec{k}$$
:::success
$$a_x = \frac{dv_x}{dt} = \frac{d^2x}{dt^2}$$

$$a_y = \frac{dv_y}{dt} = \frac{d^2y}{dt^2}$$

$$a_z = \frac{dv_z}{dt} = \frac{d^2z}{dt^2}$$
:::
